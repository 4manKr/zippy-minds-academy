Admin Guide

Full platform control — users, sessions, content, settings

Logging In

Admin login uses 3-factor authentication for security.

1
Go to Admin Login

Navigate to /auth/admin-login — this page is not shown in the public navigation.

2
Enter Email & Password

Use your admin email and password set during account creation.

3
Enter Admin Secret Key

Enter the secret key stored in your Vercel environment variables (ADMIN_SECRET_KEY). This is a one-time setup key — keep it safe.

⚠️
Never share your Admin Secret Key. If compromised, change it immediately in Vercel → Environment Variables → Redeploy.

3

Send the tutor their email and password — they log in at /auth/tutor-login

Tutor Priority System

Go to Tutors → Priority tab. Set a numeric priority for each tutor:

  • Lower number = higher priority (e.g., priority 1 gets assigned first)
  • When a parent books a demo, the system picks the tutor with the lowest priority number who is available for that slot
  • If no tutor is available, the booking is flagged as "Needs Admin" — you must manually assign

Payments & Enrollments

  • View all transactions: parent name, amount, currency (INR/USD), gateway (PayPal/Razorpay), status (Pending/Paid/Failed)

Content Management

Resources (PDFs, Documents, Links)

  • Click "Add Resource" → enter Title, Type (PDF/Image/Link), Subject, URL
  • Toggle Active/Inactive to show or hide from parents
  • Edit or delete any resource at any time

Video Lessons

Courses

Support Tickets

Go to Admin Dashboard → Support tab

  • View all tickets sorted by date — filter by Status (Open/Closed) and Priority (Low/Medium/High)
  • Click a ticket to see full message history
  • Type a reply and click "Send Reply" — parent gets an email from support@zippymindsacademy.com
  • Click "Close Ticket" once resolved — ticket moves to Closed tab
  • You can re-open a closed ticket at any time

Booking a Free Demo Session

Every new parent gets one free 30-minute demo session. Here's how to book it:

1
Go to Book Demo

Click "Book Free Demo" from the homepage, navbar, or your dashboard. You must be logged in.

2
Enter Student Details

Child's name, age, grade, and timezone (auto-detected from your browser)

3
Choose Subject

Select from: Phonics & Reading, English Grammar, Mathematics, or Public Speaking

4
Pick Date & Time

Available slots shown for the next 21 days. Grey slots are already booked. Pick any green slot.

5
Review & Confirm

Check all details, add optional notes for the tutor, then click Confirm.

✅
After booking: You'll get a confirmation email from bookings@zippymindsacademy.com. The tutor reviews and confirms, then you get a second email with your Zoom link.

Accepting or Rejecting a Demo Request

1
Go to Requests tab

New requests show student name, subject, grade, date, time, and parent info

2
Review the request

Check you're available at the requested time

3
Click Accept or Decline

If accepted: Zoom link is generated, parent is notified. If declined: admin is notified and assigns another tutor.

⏰
Respond quickly! Parents are waiting for confirmation. Try to accept or reject within a few hours of receiving the request email.

Starting a Session

  • Go to the Upcoming tab and find your session
  • Click the green "Start Zoom Meeting" button (your host link)
  • This link is different from the parent's join link — only you have the host link
  • You'll get a reminder email 30 minutes before at your registered email

Adding Session Notes

  • Click on any session card to expand it
  • Type your notes in the Notes field (what was covered, homework, next steps)
  • Click Save — notes are visible to the admin

Setting Your Availability

Go to Dashboard → Availability tab

You manage availability per month to help the admin assign you to the right slots.

1
Select a Month

Use the 3-month picker to select the month you want to set availability for

2
Select Time Slots

For each day (Mon–Sun), tick the time slots when you're available. Slots run 9:00 AM to 7:00 PM.

3
Save Availability

Click Save. The admin can now see your availability and assign sessions to your free slots.

🔒
Locked slots (shown in grey) are already booked — you cannot remove these. To change a booked slot, contact your admin who will initiate a change request.

Responding to Availability Change Requests

If the admin needs to change your availability for an already-booked slot:

  • Affected parents receive an approval email automatically
  • Once all parents approve, the admin applies the change
  • You receive a notification when the change is applied

Tips & Common Questions

For Parents

  • Use Google login for the fastest experience — no codes to remember
  • Check your spam folder if you don't receive OTP codes within 2 minutes
  • The Zoom link appears on your session card once the tutor confirms — check your dashboard
  • Book your demo at least 24–48 hours in advance for best tutor availability
  • Your timezone is auto-detected — double check it before confirming

For Tutors

  • Set your monthly availability before the start of each month so you receive demo requests
  • Accept or decline demo requests as quickly as possible — parents are waiting
  • Your Zoom host link (Start Meeting) is different from the parent's join link — use only your link
  • Upload session materials after each class to help students revise
  • Contact admin if you need to cancel a confirmed session — don't just not show up

For Admins

  • Check "Needs Admin" sessions daily — these need manual tutor assignment
  • Approve new tutor applications promptly — pending tutors cannot receive sessions
  • Use Maintenance Mode before making major changes — disables site for non-admins immediately
  • Settings changes (email, phone) take effect site-wide immediately — no redeploy needed
  • The priority system auto-assigns tutors — keep priorities updated to manage tutor workload
